\UnnumberedChapter{PHỤ LỤC}

\section*{Phụ lục 1. Hình ảnh minh họa}

Sinh viên chèn hình ảnh minh họa, ảnh chụp màn hình sản phẩm hoặc giao diện hệ thống nếu được phép công bố.

\section*{Phụ lục 2. Tài liệu, mã nguồn hoặc liên kết liên quan}

Sinh viên có thể bổ sung:

\begin{itemize}
    \item Link GitHub/GitLab nếu được phép.
    \item Tài liệu thiết kế.
    \item Tài liệu hướng dẫn sử dụng.
    \item File cài đặt hoặc tài liệu bàn giao.
\end{itemize}
